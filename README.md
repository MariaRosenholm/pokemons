# Pokedex

Business College Helsinki school assignment with React function components with hooks: useEffect, useState. Axios for fetching data from RESTful API https://pokeapi.co/.

## Build with

- React
- Axios
- https://pokeapi.co/
- Bootstrap
- React Router 6
